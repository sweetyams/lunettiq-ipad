# Change Tracking

At the end of every session that modifies code, update `CHANGELOG.md` with:

```markdown
## [Date] — Summary

### Added
- New screen: Client profile (3-column iPad layout)

### Changed
- Updated API client to handle 401 re-auth

### Fixed
- Photo upload queue draining in wrong order
```

Keep entries concise. One line per change. Group by type.
